# Anugrah K S – Personal Portfolio

Welcome to the official source code of my interactive developer portfolio – [anugrahks.dev](https://anugrahks.dev), built with [Astro](https://astro.build), [Three.js](https://threejs.org/), and [GSAP](https://gsap.com/).

This is a premium, interactive grid-based terminal and mesh developer portfolio showcasing creative frontend engineering, modern styling, and clean modular structures.

---

## 🚀 Getting Started

To run the project locally on your machine:

```bash
# 1. Clone this repository
git clone https://github.com/anugrahks19/My_Portfolio.git
cd My_Portfolio

# 2. Install package dependencies
npm install

# 3. Spin up the development server
npm run dev
```

The application will run locally at `http://localhost:4321/`.

---

## 🛠️ Tech Stack & Architecture

- **Framework:** [Astro](https://astro.build/) (Static Site Generation for hyper-fast loads)
- **Styling:** Modular SASS (SCSS) using modern CSS variables
- **Animations:** [GSAP (GreenSock Animation Platform)](https://gsap.com/) for timeline coordinates & intro stroke drawing
- **Interactive Meshes:** Custom WebGL elements built with [Three.js](https://threejs.org/) for snappy, responsive grid interactivity
- **Smooth Scrolling:** [Lenis](https://lenis.darkroom.engineering/) for natural feeling scroll deceleration
